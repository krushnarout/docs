A common transistor. You can customize the transistor type with the `transistorType` prop.


## Example Usage

```tsx ExampleTransistorUsage.tsx
  export const ExampleTransistorUsage = () => (
  <transistor
    name="Q1"
    transistorType="npn"
    footprint="sot23"
  />
)
```
