A common LED (Light Emitting Diode) component. LEDs are polarized components, with a positive (anode) and negative (cathode) terminal.

## Props

- **`pos`**: The positive terminal (anode) of the LED.
- **`neg`**: The negative terminal (cathode) of the LED.

## Example Usage

### Simple Example
```tsx ExampleLedUsage.tsx
export const ExampleLedUsage = () => (
  <led
    name="LED1"
    color="red"
    pos="net.VCC"
    neg="net.GND"
    footprint="0603"
  />
)
